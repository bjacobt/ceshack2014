Unity

Unity is a platform that manages different devices. It also provides a robust mechanism to transport data from sensors to the cloud, announce device presence.

Two kinds of devices exist today, sensors and appliances. Sensors collect data from environment and publish in the Unity network. Appliances and or other sensors can listen to these data and act upon it. Appliances are different Virtual Machines that like Analytics, Database, etc.

One such appliance is a Web Socket bridge which takes the data published by sensors and makes it available to web developers. The web socket bridge runs node.js with Socket.io which is a Javascript library for real-time web applications.
